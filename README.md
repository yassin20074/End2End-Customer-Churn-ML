# End2End Customer Churn Ml 🚀

**NeuroChurn AI** is an **End-to-End Machine Learning system** for predicting customer churn with:

- **Complete Data Pipeline** (Preprocessing + Feature Engineering)  
- **ML Model** for churn prediction (RandomForest / Logistic Regression)  
- **FastAPI Backend** for API-based predictions  
- **Interactive Streamlit Dashboard** for data input and results visualization  

---

# 🌟 Features

- Comprehensive preprocessing (handling missing values, encoding, scaling)  
- Feature Engineering for both categorical and numerical features  
- Full ML pipeline: Train → Evaluate → Predict  
- API endpoint for predictions (`/predict`)  
- Streamlit Dashboard for user interaction  
- Logging of all prediction sessions  

---

# 💻 Tech Stack

- Python 3.12  
- pandas, numpy, scikit-learn  
- FastAPI  
- Streamlit

# 🎯 Usage
- Open the Streamlit Dashboard
- Enter customer details (tenure, MonthlyCharges, TotalCharges, Contract type, Payment method)
- Click Predict
- View the Churn Probability and Predicted Class (Churn / No Churn)

#  📊 Model Evaluation
- RandomForestClassifier / Logistic Regression
- Metrics: Accuracy, Precision, Recall, F1-score
- Confusion Matrix / ROC Curve visualization possible
